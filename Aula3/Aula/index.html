1. Color

    2. Responsável por mudar a cor de um texto

    Exemplo: style="color:blue"
  2. Background-color

    3. Muda a cor do fundo do componente

    Exemplo: style="background-color:rgb(0, 255, 21)"
  3. Position

    4. Define o método que o componente vai reagir de acordo com os outros componentes. Exemplo: “position: absolute” não é afetado por nenhum componente e não afeta nenhum componente.
  4. Z-index

    5. É o índice que define se um componente vai ficar sobreposto ao outro
  5. Top

    6. Define a distância do topo de um componente dentro de outro componente.
  6. Right

    7. Define a distância do lado direito de um componente dentro de outro componente
  7. Left

    8. Define a distância do lado esquerdo de um componente dentro de outro componente.
  8. Bottom

    9. Define a distância do fundo de um componente dentro de outro componente.
  9. Width

    10. Define a largura de um componente.

    Exemplo: style="background-color:rgb(0, 255, 21)"
  10. Height

    11. Define a altura de um componente.
  11. Flex

    12. Define de acordo com um número a disponibilização de um componente entre os outros componentes.
  12. Flex-direction

    13. Muda o formato que vários componentes se comportam dentro de outro. Exemplo: “flex-direction: column” irá fazer com que os componentes sejam alinhados de acordo como colunas.
  13. Font-size

    14. Define o tamanho da fonte do texto.
  14. Margin

    15. Define a área de margem em todos os lados de um elemento.
  15. Padding

    16. Define a área de preenchimento em todos os lados de um elemento de uma só vez
  16. Border

    17. Define uma borda ao redor do elemento
  17. Overflow

    18. Permite um componente ultrapassar das bordas de outro componente
  18. Text-align

    19. Define como o texto será alinhado dentro do componente
  19. Justify-content

    20. Define de acordo com o axis (x) da página onde os componentes irão começar. Exemplo: “justify-content: start” irá fazer com que os componentes sejam ordenados pela esquerda.
  20. Border-radius

    21. Define o raio das bordas de um componente
  21. Display

    22. Define o layout dos componentes
  22. Text-shadow

    23. Define a sombra de um texto.
  23. Text-transform

    24. Transforma a forma como texto é exibido podendo deixar todas as letras Maiúsculas, Minúsculas, somente a primeira letra ou na largura total
